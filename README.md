A simple desktop application to download YouTube videos in MP4 or MP3 formats with selectable resolutions. Built with Python, Tkinter (GUI), pytube (YouTube stream handling), and pydub for audio conversion.

---

## Features

- Download video in MP4 at selected resolutions (1080p, 720p, 480p, 360p, highest).
- Download audio and convert to MP3.
- Simple Tkinter-based GUI for ease of use.
- Compatible with Linux/Windows/macOS (requires ffmpeg for MP3 conversion).
